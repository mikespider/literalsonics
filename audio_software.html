Magnetic Field → Sound

Magnetic Field → Sound is an experimental sonification tool that translates real-time geomagnetic activity into an evolving audio experience. Built around live data from the NOAA Space Weather Prediction Center, the system uses the planetary K-index (Kp)—a global measure of disturbances in Earth’s magnetic field—to drive a multi-voice synthesizer.

Rather than visualizing the data through graphs, the application renders it as sound, allowing users to perceive geomagnetic dynamics through listening.

The core of the system is a three-voice structure:

The first voice maps geomagnetic intensity (Kp) to pitch, while changes in the index influence amplitude, creating a stable tonal layer.

The second voice inverts this relationship, emphasizing fluctuations and instability.

The third voice introduces additional motion and contrast, enriching the sonic texture.

Together, these voices create a dynamic interplay between stability and change. Quiet geomagnetic periods produce restrained tones, while disturbances generate complex and evolving sonic behavior.

The interface allows waveform selection, octave control, volume shaping, and inter-voice modulation, turning scientific data into a playable instrument.

This project explores perception over visualization, transforming invisible planetary forces into an audible experience.

Whale Migration Sonifier

Whale Migration Sonifier is an advanced audio plugin that converts real scientific whale migration data into dynamic, evolving soundscapes. By mapping latitude and longitude from tracked whales across the globe, the plugin transforms natural movement into expressive musical structures.

At its core, the system turns real-world data into modulation — allowing you to hear migration patterns as sound.

🎛️ Sound Engine

A root oscillator defines the tonal center, while additional oscillators map latitude and longitude into continuously shifting pitch. The result is a living composition driven entirely by real whale movement.

🎚️ Scale Quantizer & Musical Control

Switch between raw chromatic mapping or constrain output to musical scales. Select tonal centers and scale types to balance scientific accuracy with musicality.

🌍 Dataset Control

Explore short datasets for detail or long datasets for large-scale migration patterns. Playback speed allows real-time time compression or expansion.

🔊 Synthesis & Modulation

Multi-oscillator engine, filters, delay, reverb, ADSR envelopes, and LFO modulation with assignable targets enable deep sound design control.

Every sound is derived from real whale tracking data, revealing hidden structures in migration behavior.

🎧 Applications

Ambient composition, cinematic sound design, experimental music, and scientific exploration.

Morse Code Sonifier

Morse Code Sonifier is a text-to-sound instrument that transforms written language into structured sonic patterns. It converts text into Morse code and uses these signals to drive a multi-voice synthesis engine.

Rather than simple tones, it interprets text as rhythm and time, creating evolving textures from communication itself.

How It Works

Characters are translated into dots and dashes, triggering oscillators and envelopes. Short phrases create rhythmic pulses, while longer texts generate complex soundscapes.

Multi-Voice Engine

Each oscillator features waveform selection, ADSR envelopes, and independent control, enabling rich layering.

Advanced Modulation

Flexible LFO system with tempo sync, scale quantization, and per-voice routing allows deep modulation and harmonic coherence.

Effects

Reverb, delay, and stereo shaping transform signal patterns into immersive environments.

Concept

This instrument bridges communication systems, algorithmic music, and modular synthesis — turning encoded information into expressive sound.

It is not just a synthesizer, but a translation engine where meaning becomes rhythm, and rhythm becomes music.
