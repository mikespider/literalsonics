Magnetic Field → Sound

Magnetic Field → Sound translates real-time geomagnetic activity into sound using live planetary data. Instead of visual graphs, the system allows users to perceive magnetic fluctuations through listening.

Built around a three-voice architecture, the system balances stability and variation, producing evolving sonic textures driven entirely by Earth's magnetic behavior.

Through modulation, waveform control, and synthesis parameters, it becomes both an analytical tool and a performative instrument.
