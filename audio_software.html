Our Software

Magnetic Field → Sound

An experimental sonification tool that translates real-time geomagnetic activity into an evolving audio experience. Built around live data from the NOAA Space Weather Prediction Center, it uses the planetary K-index (Kp) to drive a multi-voice synthesizer. Instead of graphs, the application renders data as sound for intuitive listening.

Whale Migration Sonifier

Converts real whale migration data into dynamic, evolving soundscapes. Latitude and longitude are mapped to pitch, producing a living composition. Features multi-oscillator synthesis, filter, delay, reverb, LFO modulation, and scale quantizer for scientific or musical exploration.
